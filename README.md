# React와 SpringBoot를 활용한 게시판 사이트
## 초기 목표
백엔드 개발의 기초를 잡기 위해 
- 로그인, 회원가입
- 게시판 등록, 수정, 삭제, 조회
- 게시판 댓글
- 일반적인 댓글이 가능하면 실시간 댓글 확인
위의 기능들을 구현할 생각으로 프로젝트를 시작.

기본 목적이 백엔드 기술의 학습이기 때문에 프론트 디자인은 신경쓰지 않기로 함.

## 사용 기술
### React<br>
React를 선택한 이유: 컴포터는트의 재사용성, 화면의 깜빡임 없이 변경되는 UI
### SpringBoot<br>
SpringBoot를 선택한 이유: 내장 서버 제공으로 인한 빠른 서버실행, 쉬운 의존성 관리
### PostgreSQL<br>
PostgreSQL을 선택한 이유: 무료.

## 현재 개발된 기능
전체적으로 axios 라이브러리를 이용해 Spring과 JSON으로 통신 중
### 로그인, 회원가입
- Spring Security를 이용해 비밀번호 암호화, 사용자 인가
### 게시판 등록, 수정, 삭제, 조회
- Jpa를 이용해 테이블을 만들고 각종 데이터들을 쉽게 조작
